== NAME

dve2lts-sym - Symbolic reachability analysis for DiVinE DVE models.

== SYNOPSIS

*{manname}* [OPTION]... 'input'<_.dve_|_.dve2C_> ['output.etf']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
in 'input.dve/input.dve2C'.  Specifications are in _dve_ format, or in
_dve2C_ format generated by the divine tool.  'output.etf'
specifies the name of the output manpage:etf[5].
