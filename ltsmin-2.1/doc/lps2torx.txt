== NAME

lps2torx - TorX RPC interface for LPS models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.lps'

== DESCRIPTION

{manname} provides access to a labelled transition system from a
specification provided in 'input.lps' via the TorX-Explorer textual
interface on _stdin_ and _stdout_.  Specifications are in _lps_ format
and are commonly generated by manpage:mcrl22lps[1].
