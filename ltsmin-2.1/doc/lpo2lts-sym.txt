== NAME

lpo2lts-sym - Symbolic reachability analysis for muCRL models.

== SYNOPSIS

*{manname}* [OPTION]... 'file.tbf' ['outfile.etf']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
in 'file.tbf'.  Specifications are in _tbf_ format and are commonly
generated by manpage:mcrl[1] or manpage:mcrl22mcrl[1].  'outfile.etf'
specifies the name of the output archive, written in manpage:etf[5]
format.
