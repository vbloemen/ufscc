== NAME

pins2lts-seq - Sequential state space generator for plugin models.

== SYNOPSIS

*{manname}* --loader=<plugin> [OPTION]... <input> ['output.fmt']
*{manname}* [OPTION]... <model.so> ['output.fmt']

== DESCRIPTION

{manname} generates a labelled transition system from a specification
provided as a plugin/input pair by enumerative reachability analysis using a
general state expanding algorithm. 'output.fmt'
specifies the name of the output archive.  The desired format is
deduced from the filename extension. Available formats are described
below.
