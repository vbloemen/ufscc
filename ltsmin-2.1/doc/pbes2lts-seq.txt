== NAME

pbes2lts-seq - Sequential parity game generator.

== SYNOPSIS

*{manname}* [OPTION]... 'input'_.pbes_ ['output.fmt']

== DESCRIPTION

{manname} generates a parity game from a specification
provided in 'input.pbes' by enumerative reachability analysis using a
general state expanding algorithm. Specifications are in _pbes_ format
and are commonly generated by manpage:lps2pbes[1].
'output.fmt' specifies the name of
the output archive.  The desired format is deduced from the filename
extension. Available formats are described below.
