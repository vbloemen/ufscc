== NAME

lps2lts-sym - Symbolic reachability analysis for mCRL2 models.

== SYNOPSIS

*{manname}* [OPTION]... 'input.lps' ['output.etf']

== DESCRIPTION

{manname} performs a reachability analysis on a specification provided
in 'input.lps'.  Specifications are in _lps_ format and are commonly
generated by manpage:mcrl22lps[1].  'output.etf' specifies the name
of the output archive, written in manpage:etf[5] format.
